Responsive Sass version 0.1.0

Download on GitHub

Use:

Install (this gem currently requires the Compass gem)

gem install responsive-sass

Import Responsive Sass into your project.

@import "responsive-sass";

This gems core provides standard @media queries for you to easily make your project responsive.

Each mixin comes with these options: 

width, height, background, display, font-size, margin

Setup your element with the @media tag[s] needed for your view.

section {@include mobile-portrait(20em, $margin: $section-margin-top auto);}

You can the Frameless grid like this:

section {@include mobile-portrait($col4);}

Here is an example of how this site is using Responsive Sass:

// Drying up my mixins using Sass @extend
        section, footer {
         @extend .mobile-portrait;
         @extend .mobile-landscape;
        }
        // Classes to extend
        .mobile-portrait {
         @include mobile-portrait($cols4);
        }                                
        .mobile-landscape {
         @include mobile-landscape($cols6);
        }
